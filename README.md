## Pre-requisits (FLTK installation):

###  commands on Linux
```Bash
sudo apt-get install fltk1.3-dev
sudo apt-get install libasound2-dev 
```
### commands on Mac
```Bash
sudo brew install fltk
```


## Compilation:

### Building on Linux
```Bash
make linux
```
### Building on Mac
```Bash
make mac
```
